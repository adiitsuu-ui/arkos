Overview

What is Arkos?

A full peer-to-peer electronic cash system written in Rust. No pre-mine, no VC allocation, no special hardware advantage. Every full node validates the same rules; every miner competes on equal footing.

⚡

Neural Proof of Work

ArkHash runs a 16-layer INT8 fully-connected chain — the exact primitive on every modern phone's NPU. Mining accelerates on Apple ANE, Qualcomm Hexagon, MediaTek APU, and Google Tensor without any special protocol privileges.

🔐

Post-Quantum Signatures

Every transaction carries both a classical ECDSA signature and a CRYSTALS-Dilithium (NIST FIPS 204) lattice-based signature. Both must independently verify. Breaking one is not enough to forge a transaction.

🔒

Encrypted at Every Layer

Vault keys are wrapped in AES-256-GCM derived via Argon2id. All P2P traffic uses Noise_XX with mutual authentication and forward secrecy. Your private keys never leave your machine in plaintext.

π

Pi-Capped Supply

Exactly 31,415,926 ARKOS will ever exist — π × 10⁷. Named after the Greek arché, sharing its root with Archimedes, the mathematician who gave humanity Pi. The chain enforces the cap in consensus; no node will accept a block that exceeds it.

🌐

Open Network

Any device can connect, mine, and submit transactions. No attestation, no hardware registration, no platform check. The protocol magic separates mainnet, testnet, and regtest — pick the network and connect.

🦀

Memory-Safe by Default

Written entirely in Rust. Key material is erased from memory on drop using the Zeroize crate. Buffer overflows, use-after-free, and data races are eliminated at compile time.

Consensus

ArkHash — Neural Proof of Work

ArkHash is a proof-of-work function whose core is a chain of INT8 fully-connected layers — the universal primitive on every modern mobile and laptop Neural Processing Unit. It democratises mining across billions of existing consumer devices.

Step 1 seed = SHA256(header_bytes ‖ nonce_le64) // 32-byte seed from block header + nonce
Step 2 x₀ = expand_seed(seed) → [i8; 256] // 256-dimensional INT8 activation vector
Step 3 for l in 0..16: // 16 sequential FC layers
3a y = W_l · x_l + b_l // INT8×INT8→INT32 dot product
3b x_{l+1} = clamp(y >> 12, −127, 127) // scale back to INT8
Step 4 result = SHA256(x_16 as bytes) // one-way output gate
Check valid = result ≤ bits_to_target(bits) // standard PoW difficulty check

The 1 MB weight table (16 × 256 × 256 INT8 bytes) is deterministically derived from a protocol constant using SHA256 in counter mode. It fits entirely in the on-chip SRAM of every modern NPU, eliminating DRAM bandwidth as a bottleneck.

Security

Post-Quantum by Design

Arkos uses a hybrid signature scheme on every transaction. Both signatures must independently verify. Compromising one scheme — classical or quantum — is not enough to forge a transaction.

Get Started

Run Arkos in Minutes

Arkos requires Rust. Install it via rustup.rs, then:

1

Build

Compile the release binary from source.

# clone and build
cargo build --release
2

Initialise

Create your master key and first wallet.

arkos init
# enter a passphrase (≥ 12 chars)
# saves vault to ~/.arkos/vault.enc
3

Mine

Mine your first block to your wallet address.

arkos mine --address <YOUR_ADDRESS>
arkos info
4

Run a Node

Start a full node and connect peers.

arkos node \
  --network testnet \
  --miner <ADDRESS> \
  --rpc-token <TOKEN>
5

Send ARKOS

Transfer coins to another address.

arkos send \
  --from-label "default-wallet" \
  --to <RECIPIENT> \
  --amount 5000000000
# 5000000000 arkes = 5 ARKOS
6

Full Demo

Run everything end-to-end in one command.

cargo run --release \
  -- demo
# wallets, mining, transactions,
# and security — all at once
